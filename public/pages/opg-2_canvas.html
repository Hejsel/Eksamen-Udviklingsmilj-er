HTML Canvas

Hvad er HTML Canvas?

HTML-elementet <canvas> er et kraftfuldt værktøj, der bruges til at tegne grafik og skabe animationer direkte i browseren. Det fungerer ved at tillade JavaScript at manipulere individuelle pixels, hvilket giver høj fleksibilitet.

Hvordan fungerer Canvas?

Canvas-elementet bruges primært til dynamisk rendering af grafik og animationer via JavaScript. Den centrale metode er at få adgang til en tegningskontekst, som tilbyder et API til at tegne grundlæggende former, billeder og til at manipulere pixels direkte. Canvas og JavaScript Eksempler

Introduktion til Canvas

Canvas-elementet er en bitmap-baseret grafikflade, der tillader lav-niveau grafisk rendering direkte i browseren. Det bruges især til at skabe to-dimensionelle tegninger (2D-grafik) og til mere avancerede grafiske operationer som animationer, billedmanipulation og interaktive brugeroplevelser.

For at forstå canvas, kan du forestille dig det som et tomt maleri-lærred, hvor du (gennem JavaScript) er kunstneren, der skaber alt fra simple geometriske former til komplekse interaktive scener.

Fagbegreber:

  • Bitmap
  • 2D-grafik
  • Rendering
  • Grafikflade

Billedsprog:

Canvas er som en digital version af en kunstners lærred. Hver pixel er som en penselstrøg, som du styrer gennem kode.

1. Oprettelse af Canvas-element

Canvas-elementet oprettes i HTML ved hjælp af <canvas>-tagget. For at sikre korrekt visning, defineres elementets dimensioner enten direkte som attributter (width og height) eller via CSS-styling. Hvis dimensionerne ikke defineres, har canvas en standardstørrelse på 300x150 pixels. 

    <canvas id="myCanvas" width="500" height="400"></canvas>
Læs mere

Her:

  • id bruges til at identificere canvas i JavaScript.
  • width og height definerer lærredets fysiske dimensioner, som påvirker grafikkens skarphed og layout.

Det er vigtigt at huske, at canvas-elementet i sig selv ikke indeholder nogen grafik eller interaktivitet – det er udelukkende en beholder.

Fagbegreber:

  • Attributter
  • Dimensioner
  • Skarphed
  • Standardstørrelse

Billedsprog:

Canvas-elementet er som at sætte rammen på dit maleri – du fastlægger, hvor stort lærredet skal være, inden du begynder at male.

2. Adgang til Canvas med JavaScript

Når canvas-elementet er oprettet, skal du bruge JavaScript til at få adgang til dets tegningskontekst. Tegningskonteksten er et objekt, der leverer et API (Application Programming Interface) til at tegne grafik. Der findes to typer af kontekster: 2D (for flade tegninger) og WebGL (for 3D-rendering). 

    const canvas = document.getElementById("myCanvas");\nconst ctx = canvas.getContext("2d");
Læs mere

Her:

  • document.getElementById() bruges til at finde canvas-elementet i DOM'en (Document Object Model).
  • getContext("2d") returnerer en 2D-tegningskontekst, som giver adgang til metoder som fillRect() og arc().

Tegningskonteksten kan ses som værktøjskassen, der indeholder alle pensler og farver, du har brug for til at tegne på lærredet.

Fagbegreber:

  • DOM
  • Tegningskontekst
  • API
  • 2D
  • WebGL

Billedsprog:

Tegningskonteksten er som en samling af malerpensler, farvepaletter og linealer, der gør det muligt at skabe grafiske mesterværker.

3. Tegning af Grundlæggende Figurer

Når du har adgang til konteksten, kan du begynde at tegne grundlæggende figurer som rektangler, cirkler og linjer. Her lærer vi om stregtegning (stroke), udfyldning (fill) og paths i canvas.

Fyldt Rektangel

Et fyldt rektangel oprettes med metoden fillRect(). Her definerer vi farven med fillStyle som "blue" for at vise en tydelig kontrast på canvas.

Rektanglets position bestemmes af:

  • x: 50 (vandret startposition).
  • y: 50 (lodret startposition).
  • bredde: 100 (rektanglets bredde).
  • højde: 100 (rektanglets højde).
ctx.fillStyle = "blue";
ctx.fillRect(50, 50, 100, 100);

Omrids af Rektangel

Et omrids af et rektangel oprettes med metoden strokeRect(). Farven defineres med strokeStyle som "red".

Omrids bruges ofte til at fremhæve en grænse eller skabe en kontrast. Her er positionen forskudt fra det fyldte rektangel for at vise forskellen.

  • x: 200.
  • y: 50.
  • bredde: 100.
  • højde: 100.
ctx.strokeStyle = "red";
ctx.strokeRect(200, 50, 100, 100);

Cirkler og Buede Figurer

En cirkel oprettes med metoden arc(), som er en del af en path. Vi bruger beginPath() for at starte en ny tegning.

Cirklens position og form defineres af:

  • x: 150 (centrum for cirklen).
  • y: 150.
  • radius: 50.
  • startvinkel: 0.
  • slutvinkel: 2 * Math.PI (fuld cirkel).

Vi udfylder cirklen med fill() og tilføjer en streg med stroke().

ctx.beginPath();
ctx.arc(150, 150, 50, 0, 2 * Math.PI);
ctx.fillStyle = "green";
ctx.fill();
ctx.stroke();

Linjer

En linje oprettes med moveTo() (startpunkt) og lineTo() (slutpunkt). Linjer er en del af en path.

Farven sættes med strokeStyle, og tykkelsen justeres med lineWidth.

  • startpunkt: (10, 10).
  • slutpunkt: (300, 150).
  • farve: purple.
  • tykkelse: 3.
ctx.beginPath();
ctx.moveTo(10, 10); // Startpunkt
ctx.lineTo(300, 150); // Slutpunkt
ctx.strokeStyle = "purple";
ctx.lineWidth = 3;
ctx.stroke();

4. Tekst

Canvas giver mulighed for at tilføje tekst med forskellige stilarter. Her kan vi vise fyldt tekst og omrids af tekst.

Fyldt Tekst

Metoden fillText() bruges til at tegne tekst med fyldfarve. Tekstens skrifttype og størrelse sættes med font. 

ctx.font = "20px Arial";
ctx.fillStyle = "black";
ctx.fillText("Hej Canvas!", 50, 50);

Omrids af Tekst

Metoden strokeText() tegner kun teksten som omrids, hvilket skaber en visuel kontrast. 

ctx.strokeText("Hej Canvas!", 50, 80);
Yderligere Forklaring

Forklaring:

Canvas giver dig mulighed for at tilføje tekst med forskellige stilarter. Metoden fillText() bruges til at tegne tekst med fyldfarve, og strokeText() bruges til kun at tegne tekstens omrids. Du kan ændre skrifttype og størrelse med font.

Billedsprog:

Forestil dig, at du skriver bogstaver med en pensel. Fyldt tekst er som farvede bogstaver, mens omrids er som bogstaver tegnet med en fin streg.

Fagbegreber:

  • fillText(): Tegner tekst fyldt med farve.
  • strokeText(): Tegner kun omridset af teksten.
  • font: Angiver skrifttype og størrelse, f.eks. 20px Arial.

5. Billeder

Canvas kan bruges til at vise billeder. Metoden drawImage() tegner et billede fra en kilde til canvas.

Indlæsning af Billede

Metoden drawImage() tegner billedet på canvas. Det kræver en Image-instans, som skal indlæses først. 

const img = new Image();
img.src = "billede.jpg"; // Angiv billedstien
img.onload = () => {
  ctx.drawImage(img, 0, 0, 200, 150); // x, y, bredde, højde
};
Yderligere Forklaring

Fagbegreber:

  • drawImage(): Bruges til at placere et billede på canvas.
  • Image: Et JavaScript-objekt, der repræsenterer billedet.

Canvas fungerer som et tomt album, hvor du kan indsætte billeder. Ved hjælp af koordinater bestemmer du billedets placering og størrelse.

Yderligere Forklaring

Forklaring:

Med metoden drawImage() kan du placere billeder på canvas. Billedet skal først indlæses med en Image-instans. Når billedet er klar, kan du bestemme dets placering og størrelse på canvas.

Billedsprog:

Forestil dig canvas som en fotobog, hvor du kan indsætte billeder på specifikke sider og bestemme, hvor store eller små de skal være.

Fagbegreber:

  • drawImage(): Tegner billedet på canvas.
  • Image: Et JavaScript-objekt, der repræsenterer billedet.
  • x, y: Bestemmer billedets placering.
  • bredde og højde: Angiver billedets størrelse.

6. Animation

Animationer skabes med metoden requestAnimationFrame(), som kører en funktion rekursivt for at opdatere canvas.

Animation med Rektangel

Vi bruger clearRect() til at rydde lærredet og opdatere positionen for rektanglet ved at ændre koordinaterne i en løkke. 

let x = 0;
function animate() {
  ctx.clearRect(0, 0, canvas.width, canvas.height);
  ctx.fillStyle = "blue";
  ctx.fillRect(x, 50, 50, 50); // x, y, bredde, højde
  x += 2;
  requestAnimationFrame(animate);
}
animate();
Yderligere Forklaring

Fagbegreber:

  • requestAnimationFrame(): En metode til at lave flydende animationer.
  • clearRect(): Rydder en del af canvas, så vi kan tegne på ny.

Forestil dig animationen som en tegnefilm, hvor hver ramme er et billede. Når rammene vises hurtigt efter hinanden, ser det ud, som om figuren bevæger sig.

Yderligere Forklaring

Forklaring:

Animation i canvas opnås ved at opdatere canvas gentagne gange. Metoden requestAnimationFrame() sørger for, at opdateringen sker så glat som muligt. Før hver opdatering rydder vi canvas med clearRect(), så vi kan tegne figuren på sin nye position.

Billedsprog:

Forestil dig animationen som en tegnefilm, hvor hvert billede er en ramme. Når billederne vises hurtigt efter hinanden, ser det ud som om, figuren bevæger sig.

Fagbegreber:

  • requestAnimationFrame(): En metode, der kører en funktion i et loop for at skabe animation.
  • clearRect(): Rydder en del af canvas, så det kan opdateres.
  • koordination: Opdateres konstant for at flytte figuren.

7. Pixel-manipulation

Metoder som getImageData() og putImageData() gør det muligt at manipulere individuelle pixels i canvas.

Ændring af Pixeldata

Vi kan hente pixeldata med getImageData(), ændre dem og gemme dem tilbage med putImageData(). 

const imageData = ctx.getImageData(0, 0, width, height);
// Manipuler pixels her
ctx.putImageData(imageData, 0, 0);
Yderligere Forklaring

Forklaring:

Pixel-manipulation giver mulighed for at ændre hver enkelt pixel på canvas. Med getImageData() kan vi hente pixeldata, og med putImageData() kan vi skrive ændringerne tilbage til canvas.

Billedsprog:

Forestil dig canvas som en mosaik, hvor hver pixel er en lille flise. Ved at ændre hver enkelt flise kan vi skabe mønstre, effekter eller tilpasse farver i detaljen.

Fagbegreber:

  • getImageData(): Henter pixeldata fra en bestemt del af canvas.
  • putImageData(): Skriver ændringerne tilbage til canvas.
  • RGBA: Farveformat, hvor hver pixel består af fire værdier (Rød, Grøn, Blå, Alfa).

Alternativt Eksempel:

Vi kan ændre hver anden pixel til en grøn farve for at skabe et mønster, eller vi kan ændre bestemte farvekanaler (f.eks. fjerne rød) for at opnå en visuel effekt. 

      const imageData = ctx.getImageData(0, 0, canvas.width, canvas.height);
for (let i = 0; i < imageData.data.length; i += 4) {
  if (i % 8 === 0) { // Hver anden pixel
    imageData.data[i] = 0;   // Sæt rød kanal til 0
    imageData.data[i + 1] = 255; // Grøn kanal
    imageData.data[i + 2] = 0;   // Blå kanal
  }
}
ctx.putImageData(imageData, 0, 0);

Dette mønster viser, hvordan vi kan skabe simple visuelle effekter ved at manipulere pixeldata direkte.
