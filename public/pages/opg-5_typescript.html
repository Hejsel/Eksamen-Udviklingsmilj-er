TypeScript

Hvad er TypeScript?

TypeScript er et superset af JavaScript, der tilføjer statisk typing. Det hjælper med at opdage fejl tidligere og gør koden mere læsbar og vedligeholdelig.

TypeScript er særligt nyttigt i større projekter, hvor mange udviklere arbejder sammen, og hvor robust kode er nødvendig.

Type Inference

TypeScript kan automatisk udlede typer baseret på tildelte værdier:

let number = 5; // TypeScript ved, at dette er en 'number'
let greeting = "Hej"; // TypeScript ved, at dette er en 'string'

Komplekse Typer

TypeScript gør det muligt at definere avancerede typer, f.eks. interfaces og union-typer: 

interface User {
    id: number;
    name: string;
    email?: string; // Valgfri
}

const users: User[] = [
    { id: 1, name: "Emil" },
    { id: 2, name: "Sofie", email: "sofie@example.com" }
];

const getUserName = (user: User): string => user.name;

Hvad er TypeScript?

TypeScript er et **superset** af JavaScript, som betyder, at alt gyldigt JavaScript også er gyldigt TypeScript. TypeScript tilføjer ekstra funktioner som **statisk typesikkerhed**, **avancerede typer**, og understøtter de nyeste JavaScript-standarder.

Det kræver en kompilering til JavaScript, så det kan køre i browsere eller Node.js.

Billedsprog: Tænk på TypeScript som en sikkerhedsele, der beskytter dig mod typiske fejl i JavaScript.

Sådan fungerer TypeScript

TypeScript skal kompileres til JavaScript, før det kan køre i browsere eller Node.js. 

// Kompilering af TypeScript
tsc fil.ts

Kompileret TypeScript bliver til JavaScript:

// Genereret JavaScript-kode
var navn = "Emil";

Primitiver

Type Beskrivelse Eksempel
string Repræsenterer tekstværdier som navne. let navn: string = "Emil";
number Repræsenterer tal, både heltal og decimaltal. let alder: number = 25;
boolean Repræsenterer sandhedsværdier. let erAktiv: boolean = true;

Bemærk: Primitiver sikrer, at data kun bruges på den måde, de er defineret. For eksempel: 

// Fejl: Forkert type
navn = 42; // Type 'number' er ikke kompatibel med typen 'string'.

Arrays og Tuples

Arrays og tuples giver mulighed for at gemme lister af data. Arrays indeholder elementer af samme type, mens tuples kan have faste typer for hvert element. 

// Arrays
let navne: string[] = ["Emil", "Marie"];

// Tuples
let person: [string, number] = ["Emil", 25];

Billedsprog: Arrays er som en liste af bøger med samme genre. Tuples er som en boghylde, hvor hver hylde har en specifik type bog.

Union Types

Med union types kan en variabel have flere mulige typer. Dette bruges, når data kan have forskellig form afhængigt af situationen. 

// Union type: Kan være en streng eller et nummer
let id: string | number = 42;

// Kontrollér typen før brug
if (typeof id === "string") {
  console.log("ID er en streng: " + id.toUpperCase());
} else {
  console.log("ID er et nummer: " + (id + 1));
}

Funktionstyper

TypeScript gør det muligt at definere typer for funktionens parametre og returværdi. 

// Funktion med parametre og returværdi
function tilføj(a: number, b: number): number {
  return a + b;
}

// Brug af funktionen
console.log(tilføj(5, 10)); // Output: 15

Billedsprog: Tænk på funktionstyper som en opskrift – de definerer, hvilke ingredienser (parametre) der skal bruges, og hvad retten (returtypen) bliver.

Interfaces

Interfaces bruges til at definere strukturer for objekter og sikre, at de opfylder bestemte krav. 

// Definer et interface
interface Person {
  navn: string;
  alder: number;
}

// Brug af interfacet
let bruger: Person = { navn: "Emil", alder: 25 };

Udvidelse af interfaces: Interfaces kan udvides for at tilføje yderligere egenskaber. 

interface Ansat extends Person {
  titel: string;
}

let ansat: Ansat = { navn: "Marie", alder: 30, titel: "Udvikler" };

Opsætning og Kompilering

  1. Installer TypeScript: npm install -g typescript
  2. Opret en tsconfig.json fil:
{
    "compilerOptions": {
        "target": "ES6",
        "module": "commonjs",
        "strict": true
    }
}

Kompilér TypeScript med tsc kommandoen.

TypeScript: Primitiver

Primitiver er de mest grundlæggende typer i TypeScript, som repræsenterer værdier som tekst, tal, og sandhedsværdier. 

// Definerer primitivtyper
let navn: string = "Emil"; // En tekststreng
let alder: number = 25; // Et tal
let erAktiv: boolean = true; // Sandhedsværdi

- string: Bruges til tekstværdier, f.eks. navne.
- number: Bruges til numeriske værdier, f.eks. alder.
- boolean: Bruges til sandhedsværdier (true eller false).

Bemærk: TypeScript sikrer, at disse typer er korrekte. Hvis du forsøger at tildele en forkert type, vil du få en fejl: 

// Fejl: Forkert type
navn = 42; // Type 'number' kan ikke tildeles typen 'string'.

TypeScript: Arrays og Tuples

Arrays og tuples bruges til at gemme lister af værdier. Arrays indeholder elementer af samme type, mens tuples kan have forskellige typer. 

// Arrays
let navne: string[] = ["Emil", "Marie"]; // Liste af strenge

// Tuples
let person: [string, number] = ["Emil", 25]; // Streng og tal

- string[]: Angiver en array, hvor alle elementer skal være strenge.
- [string, number]: Angiver en tuple, hvor første element er en streng, og andet element er et tal.

Fordel: TypeScript sikrer, at du kun tilføjer elementer af den korrekte type: 

// Fejl: Forkert type
navne.push(42); // Type 'number' er ikke tildelbar til typen 'string'.

TypeScript: Enums

Enums bruges til at definere en liste over konstante værdier med navne. De bruges ofte til at repræsentere roller, statusser eller kategorier. 

// Definerer en enum
enum Rolle {
  Admin,
  Bruger,
  Gæst,
}

// Tildeler en enum-værdi
let minRolle: Rolle = Rolle.Admin; // Admin er 0

- enum: Opretter en navngivet samling af konstante værdier.
- Rolle.Admin: Enum-værdien repræsenteret af `Admin`.

Bemærk: TypeScript tildeler som standard numeriske værdier til enums, startende fra 0: 

console.log(Rolle.Admin); // Output: 0
console.log(Rolle.Bruger); // Output: 1

TypeScript: Union Types

Union Types bruges til at tillade flere typer for en variabel. Det betyder, at en variabel kan være én af flere angivne typer. 

// Union type: Kan være en streng eller et nummer
let id: string | number = 42;

// Tjek typen før brug
if (typeof id === "string") {
  console.log("ID er en streng: " + id.toUpperCase());
} else {
  console.log("ID er et nummer: " + (id + 1));
}

- |: Bruges til at angive en union type.
- typeof: Bruges til at finde ud af typen ved runtime.

TypeScript: Funktionstyper

Funktionstyper definerer typerne for en funktions argumenter og returværdi. 

// Funktion med typer for argumenter og returværdi
function tilføj(a: number, b: number): number {
  return a + b;
}

// Kald funktionen
console.log(tilføj(5, 10)); // Output: 15

- a: number: Angiver, at argumentet `a` skal være et tal.
- : number: Angiver, at funktionen returnerer et tal.

Bemærk: Hvis du forsøger at bruge forkerte typer, vil TypeScript give en fejl: 

// Fejl: Forkert type
tilføj("5", 10); // Argumentet af typen 'string' er ikke tildelbart til typen 'number'.

Hvad er TypeScript?

TypeScript er et programmeringssprog, der bygger oven på JavaScript. Det introducerer **statisk typesikkerhed**, hvilket betyder, at du kan definere, hvilke typer data variabler, funktioner og objekter skal have. TypeScript gør koden mere robust, lettere at vedligeholde og hjælper med at finde fejl under udvikling fremfor ved runtime.

TypeScript bliver kompilieret til JavaScript, så det kan køre i enhver browser eller Node.js.

Primitiver

Primitiver er de grundlæggende datatyper i TypeScript. De repræsenterer standardværdier som tekst, tal og sandhedsværdier.

Type Beskrivelse Eksempel
string Repræsenterer tekstværdier som navne eller sætninger. let navn: string = "Emil";
number Bruges til numeriske værdier, både heltal og decimaltal. let alder: number = 25;
boolean Repræsenterer sandhedsværdier (true eller false). let erAktiv: boolean = true;
null Repræsenterer en tom værdi. let tom: null = null;
undefined Indikerer, at en variabel ikke har en defineret værdi. let udefineret: undefined;

Arrays og Tuples

Arrays bruges til at gemme lister af værdier med samme type, mens tuples giver mulighed for at blande typer i en struktureret liste. 

// Arrays
let navne: string[] = ["Emil", "Marie"];

// Tuples
let person: [string, number] = ["Emil", 25];

- string[]: Definerer en array, hvor alle elementer skal være strenge.
- [string, number]: En tuple med en streng og et tal i korrekt rækkefølge.

Union Types

Union Types tillader variabler at have mere end én mulig type. Dette er nyttigt, når data kan have forskellige former. 

// Union Type: Kan være en streng eller et nummer
let id: string | number = 42;

// Brug typeof til at tjekke typen
if (typeof id === "string") {
  console.log("ID er en streng: " + id.toUpperCase());
} else {
  console.log("ID er et nummer: " + (id + 1));
}

- |: Bruges til at angive, at en variabel kan være én af flere typer.

Funktionstyper

Funktionstyper definerer både typerne for funktionens argumenter og dens returværdi. 

// Funktion med typeannotationer
function tilføj(a: number, b: number): number {
  return a + b;
}

// Kald funktionen
console.log(tilføj(5, 10)); // Output: 15

- a: number: Angiver typen af argumentet `a`.
- : number: Definerer returtypen for funktionen.

Interfaces

Interfaces bruges til at definere strukturen af objekter. De gør det nemmere at specificere, hvordan data skal se ud. 

// Definerer et interface
interface Person {
  navn: string;
  alder: number;
}

// Brug interfacet
let bruger: Person = { navn: "Emil", alder: 25 };

- interface: Definerer en struktur, som et objekt skal følge.
- Person: Angiver, at et objekt skal have `navn` som en streng og `alder` som et tal.
