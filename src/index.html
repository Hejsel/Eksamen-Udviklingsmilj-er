Introduktion til Git

Git er et versionskontrolsystem, der bruges til at spore ændringer i kode og samarbejde mellem udviklere. Det giver dig mulighed for at arbejde sikkert på dit projekt uden at miste historik over dine ændringer.

Hvordan fungerer Git?

1. Initialisering af et Git Repository

Start med at køre git init i en mappe for at oprette et nyt repository.

2. Tilføjelse af Filer

Brug git add filnavn for at tilføje filer til staging-området.

3. Committe Ændringer

Gem ændringer med git commit -m "Din besked" for at oprette en commit.

4. Push til Remote

Synkroniser lokalt arbejde til en fjernserver ved hjælp af git push.
