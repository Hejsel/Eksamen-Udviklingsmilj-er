Versionsstyring med Git og Gruppearbejde

Præsenter Git fra opsætning til generel brug. Uddyb processerne initialisering, staging og comitting. Branching og merging skal også dækkes. Inkluder cloning, pushing, pulling samt brug af et fælles serverbaseret repository.

Introduktion til Git

Git er et versionskontrolsystem, der bruges til at spore ændringer i kode og samarbejde mellem udviklere. Det giver dig mulighed for at arbejde sikkert på dit projekt uden at miste historik over dine ændringer.
