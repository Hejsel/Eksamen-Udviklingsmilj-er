Præsenter Git fra opsætning til generel brug. Uddyb processerne initialisering, staging og comitting. Fænomenerne branching og merging skal også indgå i præsentationen. Begreberne cloning, pushing og pulling samt brugen af et fælles serverbaseret repository skal detaljeres. Gennemgangen skal støttes med konkrete eksempler på Git kommandoer og deres effekt. Du bør bruge uddrag fra egne projekter og opgaver som eksempler i din gennemgang.
